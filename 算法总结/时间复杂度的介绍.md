# 时间复杂度

常见的5种时间复杂度,（log 是以2位为底的对数）

## *O* (1)

常量时间

## *O* (log *n* )

对数时间 ，这样的算法包括二分查找。

## *O* (*n* )

线性时间 ，这样的算法包括简单查找。 

## *O* (*n* * log *n* )

相关算法：快速排序

## *O* (*n* ^2 )

相关算法：选择排序

排序列表长度为n的数据，每次都遍历一遍列表（*O*（n））,需要遍历n次

## *O* (*n* !)

相关算法：旅行商问题（**旅行推销员问题**（英语：**Travelling salesman problem**, **TSP**））
