# 题目[面试题05. 替换空格](https://leetcode-cn.com/problems/ti-huan-kong-ge-lcof/)

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

示例 1：

```
输入：s = "We are happy."
输出："We%20are%20happy."
```

限制：

```
0 <= s 的长度 <= 10000
```



*****

# Python解题思路

## 方法1：使用字符串的replace

```python
class Solution:
    def replaceSpace(self, s: str) -> str:
        return s.replace(" ", "%20")
```

## 方法2：数组处理

话说也挺简单

```python
class Solution:
    def replaceSpace(self, s: str) -> str:
        result = []
        for each in s:
            if each == " ": result.append("%20")
            else: result.append(each)
        return "".join(result)
```
