# 题目[37. 解数独](https://leetcode-cn.com/problems/sudoku-solver/)

编写一个程序，通过已填充的空格来解决数独问题。

一个数独的解法需遵循如下规则：

    数字 1-9 在每一行只能出现一次。
    数字 1-9 在每一列只能出现一次。
    数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。

空白格用 '.' 表示。

一个数独。

答案被标成红色。

Note:

    给定的数独序列只包含数字 1-9 和字符 '.' 。
    你可以假设给定的数独只有唯一解。
    给定数独永远是 9x9 形式的。

# python解题思路

之前回溯的算法接触的比较少，上一次遇到是 20.有效括弧 的题目，对于回溯产生了非常好奇的感觉，

虽然回溯的算法思路听起来很简单，但是遇到具体题目需要具体分析，

特别是什么时候该回溯，什么时候该结束

```python
class Solution:
    def solveSudoku(self, board: List[List[str]]) -> None:
        """
        Do not return anything, modify board in-place instead.
        """
        answer_num = ["1", "2", "3", "4", "5", "6", "7", "8", "9"]

        box_1 = {
            "row": ["0", "1", "2"],
            "col": ["0", "1", "2"],            
        }
        box_2 = {
            "row": ["3", "4", "5"],
            "col": ["0", "1", "2"],            
        }
        box_3 = {
            "row": ["6", "7", "8"],
            "col": ["0", "1", "2"],            
        }
        box_4 = {
            "row": ["0", "1", "2"],
            "col": ["3", "4", "5"],            
        }
        box_5 = {
            "row": ["3", "4", "5"],
            "col": ["3", "4", "5"],            
        }
        box_6 = {
            "row": ["6", "7", "8"],
            "col": ["3", "4", "5"],            
        }
        box_7 = {
            "row": ["0", "1", "2"],
            "col": ["6", "7", "8"],
        }
        box_8 = {
            "row": ["3", "4", "5"],
            "col": ["6", "7", "8"],
        }
        box_9 = {
            "row": ["6", "7", "8"],
            "col": ["6", "7", "8"],            
        }
        boxs = [box_1, box_2, box_3, box_4, box_5, box_6, box_7, box_8, box_9]

        # 检查当前的board是否没有.
        def check_finish():
            for row in board:
                if "." in row:
                    return False
            return True
		
        # 找到位置对于的九空格箱子
        def find_box(row, col):
            for box in boxs:
                if str(row) in box["row"] and str(col) in box["col"]:
                    return box
        
        # 返回九空格箱子里面已经使用了的字符
        def find_box_used_num(box):
            result = []
            for row in box["row"]:
                for col in box["col"]:
                    result.append(board[int(row)][int(col)])
            return result
        
        # 判断该位置是否可以用value填充
        def check(row, col, box, value):
            # board[row] 行使用的字符 [row[col] for row in board] 列使用的字符 find_box_used_num(box)九空格箱子里面已经使用了的字符
            if value not in board[row] and value not in [row[col] for row in board] and value not in find_box_used_num(box):
                return True
            else:
                return False

        # 回溯本体函数
        def fill_num(row, col):
            # 结束标志
            if row == 8 and col == 9:
                return
            # 如果col越界就移动到下一行
            if col == 9:
                row += 1
                col = 0
            # 需要填充的位置
            if board[row][col] == ".":
                # 找到该位置对应得九空格
                box = find_box(row, col)
                # 遍历 1~9 进行尝试
                for each in answer_num:
                    # 判断是否可以使用这个值
                    if check(row, col, box, each):
                        board[row][col] = each
                        fill_num(row, col + 1)
                        # 如果没有结束
                        if check_finish() == False:
                            #走不通就会走下面这句，还原上一次走时改变的值
                            board[row][col] = "."
            # 正常的继续往右边走
            else:
                fill_num(row, col + 1)
            
        fill_num(0, 0)
```
