# 题目[146. LRU缓存机制](https://leetcode-cn.com/problems/lru-cache/)

运用你所掌握的数据结构，设计和实现一个  LRU (最近最少使用) 缓存机制。它应该支持以下操作： 获取数据 get 和 写入数据 put 。

获取数据 get(key) - 如果密钥 (key) 存在于缓存中，则获取密钥的值（总是正数），否则返回 -1。
写入数据 put(key, value) - 如果密钥不存在，则写入其数据值。当缓存容量达到上限时，它应该在写入新数据之前删除最久未使用的数据值，从而为新的数据值留出空间。

进阶:

你是否可以在 O(1) 时间复杂度内完成这两种操作？

示例:

```
LRUCache cache = new LRUCache( 2 /* 缓存容量 */ );

cache.put(1, 1);
cache.put(2, 2);
cache.get(1);       // 返回  1
cache.put(3, 3);    // 该操作会使得密钥 2 作废
cache.get(2);       // 返回 -1 (未找到)
cache.put(4, 4);    // 该操作会使得密钥 1 作废
cache.get(1);       // 返回 -1 (未找到)
cache.get(3);       // 返回  3
cache.get(4);       // 返回  4
```

## 方法1：字典

思路很简单，就是把字典用来保存数据，并且加上上次操作的时间用于判断哪个是最旧的

```python
from datetime import datetime


class LRUCache:

    def __init__(self, capacity: int):
        self.used = 0
        self.capacity = capacity
        # self.data = {
        #     "test": {
        #         "last_used_time": datetime.now(),
        #         "value": "test"
        #     }
        # }
        self.data = {}

    def get(self, key: int) -> int:
        if key in self.data.keys():
            self.data[key]["last_used_time"] = datetime.now()
            return self.data[key]["value"]
        else:
            return -1

    def put(self, key: int, value: int) -> None:
        if key in self.data.keys():
            self.data[key] = {
                "last_used_time": datetime.now(),
                "value": value
            }
        else:
            if self.used < self.capacity:
                self.data[key] = {
                    "last_used_time": datetime.now(),
                    "value": value
                }
                self.used += 1
            else:
                old_key = ""
                old_time = datetime.now()
                for each in self.data:
                    if self.data[each]["last_used_time"] <= old_time:
                        old_key = each
                        old_time = self.data[each]["last_used_time"]
                del self.data[old_key]
                self.data[key] = {
                    "last_used_time": datetime.now(),
                    "value": value
                }
                self.used += 1
```

## 方法2：队列

```python
class LRUCache:

    def __init__(self, capacity: int):
        self.capacity = capacity
        self.queue = []

    def get(self, key: int) -> int:
        result_value = -1
        result_index = -1
        for index in range(len(self.queue)):
            if self.queue[index][0] == key:
                result_value = self.queue[index][1]
                result_index = index
        if result_value != -1:
            self.queue.pop(result_index)
            self.queue.append((key, result_value))
        return result_value

    def put(self, key: int, value: int) -> None:
        result_index = -1
        for index in range(len(self.queue)):
            if self.queue[index][0] == key:
                result_index = index
        # 是否有找到
        if result_index != -1:
            self.queue.pop(result_index)
            self.queue.append((key, value))
        else:
            if len(self.queue) == self.capacity:
                self.queue.pop(0)
            self.queue.append((key, value))
```

## 方法3：字典+队列

还是老办法，不过这次不在字典里面加入判断新旧，而是使用一个队列来保存字典key的先后顺序

```python
class LRUCache:

    def __init__(self, capacity: int):
        self.capacity = capacity
        self.key_queue = []
        self.dict = {}

    def get(self, key: int) -> int:
        if key not in self.key_queue:
            return - 1
        self.key_queue.remove(key)
        self.key_queue.append(key)
        return self.dict[key]

    def put(self, key: int, value: int) -> None:
        if key in self.key_queue:
            self.key_queue.remove(key)
            self.key_queue.append(key)
            self.dict[key] = value
        else:
            if len(self.dict) == self.capacity:
                oldest_key = self.key_queue.pop(0)
                if key != oldest_key:
                    del self.dict[oldest_key]
            self.dict[key] = value
            self.key_queue.append(key)
```

```python
from collections import OrderedDict
class LRUCache(OrderedDict):

    def __init__(self, capacity):
        self.capacity = capacity

    def get(self, key):
        if key not in self:
            return - 1
        
        self.move_to_end(key)
        return self[key]

    def put(self, key, value):
        if key in self:
            self.move_to_end(key)
        self[key] = value
        if len(self) > self.capacity:
            self.popitem(last = False)
```
