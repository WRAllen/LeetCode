# 题目[2. 两数相加](https://leetcode-cn.com/problems/add-two-numbers/)

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

```
输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807
```

class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        result = []
        while True:
            l1_val = l1.val if l1 != None else 0
            l2_val = l2.val if l2 != None else 0
            mid_num = l1_val + l2_val
            if mid_num >= 10:
                if l1.next != None:
                    l1.next.val += 1
                else:
                    l1.next = ListNode(1)
                mid_num = mid_num % 10
            result.append(mid_num)
            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None
            if l1 == None and l2 == None:
                break
        new_li = LinkedList()
        for each in result:
            new_li.append(each) 
        # 用于测试
        # new_li.show()       
        return new_li.head

## 方法3：正常判断

看上面的代码，明显就有严重的重复，例如后面的l1与l2的判断，完全一样，对于这种情况其实就是变量的值问题，鉴于这里比较简单就直接加个or解决

```python
class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        head = ListNode(-1)
        tem_head = head
        while l1 and l2:
            result = l1.val + l2.val
            if result >= 10:
                result -= 10
                if l1.next: l1.next.val += 1
                else: l1.next = ListNode(1)
            tem_head.next = ListNode(result)
            tem_head = tem_head.next
            l1 = l1.next
            l2 = l2.next
        # 这里用or来解决可能是l1或者是l2的问题
        # 如果l1不为None或者False就直接赋予l3，否则对l2进行同样判断
        l3 = l1 or l2
        while l3:
            result = l3.val
            if result >= 10:
                result -= 10
                if l3.next: l3.next.val += 1
                else: l3.next = ListNode(1)
            tem_head.next = ListNode(result)
            tem_head = tem_head.next
            l3 = l3.next
        return head.next
```
